
vim.DistributedVirtualSwitch.RollbackCapability
===============================================
  The `DVSRollbackCapability <vim/DistributedVirtualSwitch/RollbackCapability.rst>`_ data object describes the rollback capabilities for a `DistributedVirtualSwitch <vim/DistributedVirtualSwitch.rst>`_ .
:extends: vmodl.DynamicData_
:since: `vSphere API 5.1 <vim/version.rst#vimversionversion8>`_

Attributes:
    rollbackSupported (`bool <https://docs.python.org/2/library/stdtypes.html>`_):

       Indicates whether rollback is supported on the distributed switch.
