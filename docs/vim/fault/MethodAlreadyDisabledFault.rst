
vim.fault.MethodAlreadyDisabledFault
====================================
    :extends:

        `vmodl.RuntimeFault <vmodl/RuntimeFault.rst>`_

  A MethodAlreadyDisabledFault fault is thrown when an attempt is made to disable a method that is already disabled.
