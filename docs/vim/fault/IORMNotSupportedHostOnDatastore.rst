
vim.fault.IORMNotSupportedHostOnDatastore
=========================================
    :extends:

        `vim.fault.VimFault <vim/fault/VimFault.rst>`_

  A IORMNotSupportedHostOnDatastore fault occurs when the datastore is connected to one or more hosts that do not support storage I/O resource management.

Attributes:

    datastore (`str <https://docs.python.org/2/library/stdtypes.html>`_)

    datastoreName (`str <https://docs.python.org/2/library/stdtypes.html>`_)

    host (`str <https://docs.python.org/2/library/stdtypes.html>`_): is optional.
