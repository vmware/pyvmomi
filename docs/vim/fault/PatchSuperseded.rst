
vim.fault.PatchSuperseded
=========================
    :extends:

        `vim.fault.PatchNotApplicable <vim/fault/PatchNotApplicable.rst>`_

  This fault is thrown if a patch install fails because the patch is superseded by patches already installed.

Attributes:

    supersede (`str <https://docs.python.org/2/library/stdtypes.html>`_): is optional.
