
vim.fault.InvalidDasConfigArgument
==================================
    :extends:

        `vmodl.fault.InvalidArgument <vmodl/fault/InvalidArgument.rst>`_

  This fault is thrown when an attempt is made to configure an HA cluster with invalid argument.

Attributes:

    entry (`str <https://docs.python.org/2/library/stdtypes.html>`_): is optional.

    clusterName (`str <https://docs.python.org/2/library/stdtypes.html>`_): is optional.
