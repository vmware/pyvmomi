
vim.fault.NoCompatibleHost
==========================
    :extends:

        `vim.fault.VimFault <vim/fault/VimFault.rst>`_

  A NoCompatibleHost fault is thrown when DRS cannnot find a compatible host in a given compute resource to run a virtual machine on.

Attributes:

    host (`str <https://docs.python.org/2/library/stdtypes.html>`_): is optional.

    error (`str <https://docs.python.org/2/library/stdtypes.html>`_): is optional.
