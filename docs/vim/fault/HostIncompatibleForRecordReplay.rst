
vim.fault.HostIncompatibleForRecordReplay
=========================================
    :extends:

        `vim.fault.VimFault <vim/fault/VimFault.rst>`_

  This fault is thrown when an attempt is made record or replay a virtual machine on a host that is incompatible.

Attributes:

    hostName (`str <https://docs.python.org/2/library/stdtypes.html>`_): is optional.

    reason (`str <https://docs.python.org/2/library/stdtypes.html>`_): is optional.
