
vim.fault.ReplicationInvalidOptions
===================================
    :extends:

        `vim.fault.ReplicationFault <vim/fault/ReplicationFault.rst>`_

  A ReplicationInvalidOptions fault is thrown when the options string passed contains invalid characters or broken format.

Attributes:

    options (`str <https://docs.python.org/2/library/stdtypes.html>`_)

    entity (`str <https://docs.python.org/2/library/stdtypes.html>`_): is optional.
