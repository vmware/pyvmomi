
vim.fault.PatchBinariesNotFound
===============================
    :extends:

        `vim.fault.VimFault <vim/fault/VimFault.rst>`_

  This fault is thrown if a patch install fails because the binaries associated with the patch are not found.

Attributes:

    patchID (`str <https://docs.python.org/2/library/stdtypes.html>`_)

    binary (`str <https://docs.python.org/2/library/stdtypes.html>`_): is optional.
