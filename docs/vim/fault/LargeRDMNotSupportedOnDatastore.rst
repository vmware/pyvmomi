
vim.fault.LargeRDMNotSupportedOnDatastore
=========================================
    :extends:

        `vim.fault.VmConfigFault <vim/fault/VmConfigFault.rst>`_

  The virtual machine is configured with a 2TB+ Raw Disk Mapping. This is not supported on the datastore.
