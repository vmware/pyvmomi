
vim.dvs.EntityBackup
====================
   `EntityBackup <vim/dvs/EntityBackup.rst>`_ is an abstract data object that contains the related entity backup and restore elements for virtual distributed switches and virtual distributed portgroups. See the following elements:
   * `EntityBackupConfig <vim/dvs/EntityBackup/Config.rst>`_
   * `EntityImportType <vim/dvs/EntityBackup/ImportType.rst>`_
   * `EntityType <vim/dvs/EntityBackup/EntityType.rst>`_
:extends: vmodl.DynamicData_
:since: `vSphere API 5.1 <vim/version.rst#vimversionversion8>`_
