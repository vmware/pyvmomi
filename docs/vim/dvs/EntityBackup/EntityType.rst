vim.dvs.EntityBackup.EntityType
===============================
  The `EntityType <vim/dvs/EntityBackup/EntityType.rst>`_ enum identifies the type of entity that was exported ( `DVSManagerExportEntity_Task <vim/dvs/DistributedVirtualSwitchManager.rst#exportEntity>`_ ).
  :contained by: `vim.dvs.EntityBackup <vim/dvs/EntityBackup.rst>`_

  :type: `vim.dvs.EntityBackup.EntityType <vim/dvs/EntityBackup/EntityType.rst>`_

  :name: distributedVirtualPortgroup

values:
--------

distributedVirtualSwitch
   Indicates the exported entity is a `DistributedVirtualSwitch <vim/DistributedVirtualSwitch.rst>`_ .

distributedVirtualPortgroup
   Indicates the exported entity is a `DistributedVirtualPortgroup <vim/dvs/DistributedVirtualPortgroup.rst>`_ .
