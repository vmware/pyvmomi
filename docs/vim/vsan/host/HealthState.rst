vim.vsan.host.HealthState
=========================
  A `VsanHostHealthState <vim/vsan/host/HealthState.rst>`_ represents the state of a participating host in the VSAN service.See `VsanHostClusterStatus <vim/vsan/host/ClusterStatus.rst>`_ 

  :type: `vim.vsan.host.HealthState <vim/vsan/host/HealthState.rst>`_

  :name: unhealthy

values:
--------

healthy
   Node is considered healthy.

unknown
   Node health is unknown.

unhealthy
   Node is considered unhealthy.
