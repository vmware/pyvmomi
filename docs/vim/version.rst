vSphere API supported versions
============================================================
You may use a version's namespace to force a pyVmomi connection to use a specific version of the API.



.. _vim.version.version2:
.. _2.5:
vim.version.version2_:
2.5_:
    namespace:
        vim25/2.5
    Name:
        VI API 2.5

        vim25/2.5u2
    Name:
        VI API 2.5u2
