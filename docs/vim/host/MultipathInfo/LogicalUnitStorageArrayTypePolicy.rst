
vim.host.MultipathInfo.LogicalUnitStorageArrayTypePolicy
========================================================
  The `HostMultipathInfoLogicalUnitStorageArrayTypePolicy <vim/host/MultipathInfo/LogicalUnitStorageArrayTypePolicy.rst>`_ data object describes a storage array type policy for for a device. This policy determines how device I/O and management is performed.
:extends: vmodl.DynamicData_
:since: `vSphere API 4.0 <vim/version.rst#vimversionversion5>`_

Attributes:
    policy (`str <https://docs.python.org/2/library/stdtypes.html>`_):

       String indicating the storage array type policy.
