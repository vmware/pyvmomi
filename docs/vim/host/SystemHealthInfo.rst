vim.host.SystemHealthInfo
=========================

:extends: vmodl.DynamicData_
:since: `VI API 2.5`_

Attributes:
    numericSensorInfo (`vim.host.NumericSensorInfo`_, optional):

       Health information provided by the power probes.
