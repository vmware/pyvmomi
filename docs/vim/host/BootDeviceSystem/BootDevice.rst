
vim.host.BootDeviceSystem.BootDevice
====================================
  The `HostBootDevice <vim/host/BootDeviceSystem/BootDevice.rst>`_ data object represents a boot device on the host system.
:extends: vmodl.DynamicData_
:since: `VI API 2.5 <vim/version.rst#vimversionversion2>`_

Attributes:
    key (`str <https://docs.python.org/2/library/stdtypes.html>`_):

       The identifier for the boot device.
    description (`str <https://docs.python.org/2/library/stdtypes.html>`_):

       The description of the boot device.
